Glosario de la Unidad 6

Ambiente de seguridad

Conjunto de políticas, herramientas, procesos y tecnologías para proteger sistemas frente a amenazas y accesos no autorizados.

Aislamiento de procesos

Mecanismo que evita que un proceso interfiera o acceda a la información de otro sin autorización.

Autenticación

Verificación de identidad de un usuario antes de permitirle acceso a recursos.

Control de acceso

Mecanismo que regula quién puede ver o usar recursos computacionales.

DAC

Modelo de acceso discrecional donde el propietario decide permisos de recursos.

MAC

Modelo de acceso obligatorio basado en políticas que los usuarios no pueden modificar.

RBAC

Modelo de acceso basado en roles que asigna permisos según funciones organizacionales.

Lista de Control de Acceso (ACL)

Lista que define permisos específicos de usuarios o grupos sobre un recurso.

Hipervisor

Software que permite crear y ejecutar máquinas virtuales.

Virtualización

Tecnología que crea versiones virtuales de hardware, sistemas o redes para optimizar recursos.

Virtualización asistida por hardware

Uso de extensiones de CPU (como Intel VT-x, AMD-V) para mejorar el rendimiento de máquinas virtuales.

Emulación

Simulación de hardware o software de un sistema distinto mediante un emulador.

Paravirtualización

El SO invitado se modifica para interactuar directamente con el hipervisor mediante hypercalls.

Contenedores

Virtualización ligera que comparte el kernel del sistema anfitrión, usando herramientas como Docker.

Docker

Plataforma para crear, implementar y ejecutar aplicaciones en contenedores.

Kubernetes

Sistema de orquestación de contenedores para su despliegue y gestión automatizada.

Namespaces

Mecanismo de aislamiento entre contenedores en un sistema.

Modelo Bell-LaPadula

Modelo formal enfocado en confidencialidad con reglas como “no read up, no write down”.

Modelo Biba

Modelo formal que asegura la integridad con reglas como “no read down, no write up”.

Modelo Clark-Wilson

Modelo que mantiene integridad mediante transacciones bien definidas y separación de funciones.
